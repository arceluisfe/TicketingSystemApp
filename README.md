🎫 Ticketing System Core (C# / .NET)
Professional Overview
This project is a functional Ticketing System built using C# and .NET. It is designed to simulate the core logic of enterprise support tools (like Jira or ServiceNow), enabling the creation and management of client requests.

Key Technical Skills Demonstrated
Object-Oriented Programming (OOP): Implementation of a Ticket Model to define data structures.

C# Fundamentals: Use of properties, constructors, and string interpolation for data display.

Clean Code Architecture: Separation of the Data Model (Models/Ticket.cs) from the Application Logic (Program.cs).

Version Control Ready: Organized file structure suitable for professional GitHub repositories.

How it Works
The system initializes a new ticket object with the following parameters:

Ticket ID: Unique identifier for tracking.

Title/Subject: Concise summary of the client's issue.

Description: Detailed technical information.

Status: Defaulting to "Open" upon creation.

Timestamp: Automatic generation of the creation date and time.
